# Donation Location

- [`design_handoff_donation_location/`](design_handoff_donation_location) — the original design handoff (README, design tokens, interactive HTML prototype).
- [`app/`](app) — the Next.js implementation. See [`app/README.md`](app/README.md) for setup and run instructions.

## Quick start

```bash
cd app
npm install
npm run setup
npm run dev
```

Then open http://localhost:3000.
